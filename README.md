# 中師實小游泳池改建設計

這個 Repository 用於保存中師實小游泳池改建的概念設計、規劃文件、DXF 與手機／電腦 3D 預覽。

## 目前階段

- 專案仍處於初期探索階段，尚未進行 Design Freeze。
- 所有未經正式丈量的尺寸均為「暫估」。
- 最新版本為 **V03 屋頂位置修正版**。
- 屋頂維持單斜 10°，長邊立面向左側下降、向右側升高。
- 屋頂已由原 V02 上方位置調整至使用者標示的紅色線位置。
