# 中師實小游泳池改建設計

這個 Repository 用於保存中師實小游泳池改建的概念設計成果。

## V02 白模型

目前包含：

- 手機與電腦可操作的 3D HTML 預覽
- 3D DXF 概念量體
- 西側短邊入口與突出門廳
- 長邊落地玻璃帷幕
- 單斜玻璃屋頂，坡度 10°
- 淺端 1.2 m、深端 1.5 m、緩坡池底
- 二樓入口側量體
- 靠窗樓梯
- 二樓景觀走道
