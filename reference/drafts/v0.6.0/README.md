# v0.6.0 現行概念圖

本資料夾保存由 `GEO-0.6.0` active geometry 產生的三張獨立樓層圖與一張縱向剖面。圖面、模型、日照研究、空間圖集與 3D Viewer 均同步為 v0.6.0；v0.5.0 歷史圖保留在舊資料夾，但不再出現在現行 HTML 圖集。

2F、3F 圖保留完整 1F 的 41 × 14 m 基地／建築工作外框作淡色參照底圖，讓上層退縮、伸入與外挑可直接相對判讀；底圖不是該層樓板。

## 版本標示規則

- 檔名、SVG 的 `<title>`、頁首與圖框都必須顯示 `v0.6.0`。
- 圖框必須顯示 active revision `GEO-0.6.0` 與 `SITE-XY`；不得由檔名或日期猜測最新版。
- 每一樓層及剖面使用獨立圖號，不再以合併圖作為 0.6.0 的討論底圖。
- L3 水平旋轉 +25.5°、鏡牆外傾 +23.0°是日照工作最佳值；23° 外傾必須經結構與構造專業驗證，不是施工定案。

## 座標軸與討論標記

四張 SVG 均由 active geometry 產生；平面圖使用固定於 1F 基地的 `SITE-XY` 數值 X／Y 座標系，剖面採 X／Z 同尺度：

- 最小刻度與淡色格線為 `0.5 m`。
- 每 `2.5 m` 顯示一個數字標籤；基地端點另標示 `X41`、`Y14`。
- 座標一律寫成 `X數值／Y數值`，例如 `X2.5／Y3.5`。
- 真北與本地軸方向保留於圖面。
- 2F、3F 座標網固定對齊 1F 參照底圖，不隨 L3 旋轉。

0.5 m 是讀圖最小單位，不代表概念圖已具施工放樣精度。精確座標表仍只用於測量、結構控制點與施工放樣。

圖面可由 `npm run drawings:v060` 重現。所有圖面皆為概念協調用可編輯向量圖，不是建造、結構、施工或發包圖。
