# 日照、鏡牆能量與天花板採光分析

本文件是太陽方位、2F 水平旋轉、鏡牆外傾、有鏡／無鏡能量差、天花板採光潛力、眩光與熱效能的長期分析 owner。現行設計基準留在 03，DEC／OPEN 留在 04，模型只保存跨輸出的已核准幾何；完整方法、結果、限制與建議留在此處。

## 1. 修正後的設計目標與結論

使用者已於 `DEC-039` 澄清：鏡牆的主要目的不是增加「原本陰影區的受光面積」，而是**增加泳池實際接收的太陽能量**；不論池面原本是否已有直射，鏡面疊加的能量都必須計入。同時要求暖季不得因鏡牆增加池面能量。偏暗天花板／上部擴散面的自然光是第二效益。

本輪可成立的結論是：

1. **水平 +9.5°、鏡牆外傾 +8.5°仍是已核准的造型幾何**，但 confirmed geometry 不等於 confirmed performance。
2. **現行鏡面若全年完全暴露，不符合暖季零增量要求**。在工作材料值下，暖季池面增加 786.673 kWh（相對無鏡基準 +1.287%），冷季增加 3,445.526 kWh（+4.563%）。鏡面有效，但缺少季節閘門。
3. **固定角度本身不能完成嚴格季節選擇**。水平 −20°～+40°、外傾 −10°～+40°、每 1° 共 3,111 組掃描中，沒有任何一組同時達到「暖季 0 kWh、冷季正增益」。因此不建議為能源理由大幅改變已核准量體角度。
4. **分析已建立季節控制所需的入射窗口，但尚未核准其建築形式**：工作遮罩只讓太陽高度 ≤22°、方位 88°～135° 的光抵達鏡面。主情境暖季池面增量為 0，冷季增加 597.502 kWh（+0.791%）；9 組支點／高度敏感度均維持暖季 0，冷季增加 279.830～683.747 kWh。這些數值是後續方案要達到的性能基準，不等於已選定水平遮罩、旋轉葉片或可收納外屏。
5. **天花板／屋頂接收面有可利用潛力**：遮罩主情境冷季有 2,124.036 kWh 的反射能量落到玻璃屋頂／上部接收區，共 178 個營運時段小時；暖季為 0。這是接收面能量，不是 lux，也不能與穿透至池面的能量重複計為兩份收益。

因此，目前可確認的是兩段式控制邏輯：**鏡面角度負責把冬季陽光送往目標，另一個季節控制構件負責決定何時允許陽光抵達或離開鏡面**。水平遮陽與東側翼牆的季節採光腔、雙面旋轉葉片、可完全收納消光外屏及實體摺板均已研究，但使用者尚不滿意，依 `DEC-040` 全部維持未核准並重新開放建築方向。

## 3. 方法：真正比較「無鏡」與「有鏡」

### 3.1 無鏡基準

每個 TMY 小時先計算無鏡時落到水平池面的直達與漫射太陽能量。直達部分扣除工作 L2 體量投到池面的陰影；玻璃屋頂／玻璃牆以單一等效透射率處理。這個基準不是水溫，也沒有扣除池水反射、蒸發與設備熱損失。

### 3.2 有鏡增量

鏡面每小時可截取的直達功率為：

```text
P_intercept = DNI × A_mirror × cos(incidence)
P_pool_added = P_intercept × ρ × target_fraction × τ
```

`target_fraction` 由四角平行光束的有限幾何足跡與工作池面交集取得；同一束光是否原本已直接照到池面不再排除。逐時功率積分為 kWh：

```text
有鏡池面能量 = 無鏡池面能量 + 鏡面增量
```

反射束與 4.5° 玻璃屋頂／上部接收面的交集另外積分，作為天花板採光潛力。這一數值位於材料分光之前；若玻璃把部分能量擴散進室內，就會相應減少繼續穿透至池面的部分，兩者不得相加成重複收益。

## 4. 現行角度、全年裸露鏡面的結果

主情境：+9.5°／+8.5°、X=19 m、鏡高 3.0 m、`ρ=0.75`、`τ=0.60`。

| 指標 | 暖季 | 冷季 |
| --- | ---: | ---: |
| 無鏡池面基準 | 61,106.665 kWh | 75,512.949 kWh |
| 鏡面新增池面能量 | **786.673 kWh** | **3,445.526 kWh** |
| 有鏡池面合計 | 61,895.036 kWh | 78,958.476 kWh |
| 相對無鏡增幅 | **+1.287%** | **+4.563%** |
| 有效增加時數 | 742 h | 1,363 h |
| 單一小時最大新增功率 | 4.451 kW | 7.856 kW |
| 屋頂／上部接收能量 | 2,707.928 kWh | 9,579.310 kWh |

冷季新增能量約為暖季的 4.38 倍，證明鏡牆確實具有季節偏向；但暖季仍增加 786.673 kWh，不能符合「避免暖季增加」的要求。舊的新增受光面積為零只代表反射落在原本已有直射的位置，不能否定這個能量疊加，故不再作現行性能判定。

## 5. 固定角度能否獨立解決季節問題

以單位 `ρ=1`、`τ=1` 掃描：

- 水平旋轉：−20°～+40°，每 1°。
- 鏡牆外傾：−10°～+40°，每 1°。
- 共 3,111 組；其餘幾何固定為 X=19 m、鏡高 3.0 m。

結果沒有任何固定角度同時得到暖季 0 kWh與冷季正增益。暖冷比可藉大幅旋轉與外傾降低，例如 +40°／+40° 的比例約 0.000004，但仍不是嚴格零，而且會大幅改變已核准建築造型、接縫、樓梯與 L2 空間關係。若優先保留較高冷季收益，+40°／+15° 的暖冷比仍約 4.81%。

判讀：**太陽全年同時改變高度與方位，單一固定平面只有兩個幾何自由度，無法成為完整的季節開關。**鏡面角度負責把光送往哪裡；建築遮罩必須負責決定哪些季節與時段可以照到鏡面。

## 6. 分析用季節遮罩與未核准實體方案

### 6.1 工作遮罩

保留 +9.5°／+8.5°，在鏡面前建立可由建築構造實現的入射方向窗口：

| 條件 | 工作值 | 建築作用 |
| --- | ---: | --- |
| 最大太陽高度 | 22° | 水平百葉／深層格柵阻擋較高角度太陽 |
| 最小太陽方位 | 88° | 東側翼牆／垂直 fins 排除暖季清晨低角度光 |
| 最大太陽方位 | 135° | 西南側回折／量體深度限定有效窗口 |

這是 solar mask，不是要求機械追日。概念構造可採清水模外框內的深色水平薄翼，垂直間距約 0.45～0.60 m；以 22° 截止角估算，投影深度起始值約為間距的 `cot(22°)=2.48` 倍，即約 1.11～1.49 m。實際尺寸必須改用相對於 136.5° 鏡牆法線的 profile angle、鏡牆外傾、結構厚度及全年 3D 陰影重新定案。東側翼牆負責排除方位小於 88° 的暖季清晨光。

#### 若未來重啟可動外屏，最低維修條件

1. 馬達、控制器、接線盒與檢修口留在 L2 乾燥維修側；不得把驅動器長期暴露在泳池濕熱含氯空氣中。
2. 每月目視檢查布邊／板縫、導軌、底梁、異音、腐蝕、水漬與失準；每半年做完整開合、限位、感測器、停電復歸及手動釋放測試。
3. 至少每年由專業廠商檢查驅動、固定件、排水、張力、風鎖及電氣防護；可更換件不得被永久封在清水模內。
4. 使用清水與中性清潔劑，避免漂白水、含氯強洗劑、酸鹼、研磨及高壓水柱；濕布幕可以短暫收起，但必須儘快展開乾燥，不能長期濕捲。[Renson maintenance manual](https://dam.renson.net/m/7574462e00273d4e/original/Manual-Fixscreen-Minimal-IM7.pdf%22)
5. 正式選材前做實際泳池空氣的氯胺暴露樣板，檢查褪色、塗層黏著、脆化、軸承與異種金屬腐蝕；一般戶外保固不能取代這項驗證。
6. 控制失效模式必須先定義。尤其一般外遮陽可能在強風時自動收起，若因此在晴朗暖季暴露鏡面，就會違反本案的能源目標；不能直接沿用標準遮陽控制邏輯。

#### 本輪結論

可動雙面葉片、全收納不透光捲屏及實體摺板在技術上都能製造，但目前沒有一案同時令人滿意地解決建築美學、冬季完整鏡面效率、夏季可靠遮斷、氯胺耐久、故障安全與維修可達性。依 `DEC-040`，這些方案只作研究基線；下一輪應從新的建築概念重新出發，而不是在其中擇一深化。

## 7. 天花板與上部擴散採光

季節遮罩主情境有 2,124.036 kWh 冷季反射能量落在玻璃屋頂／上部接收區。若未來把其中 10%／20%／30% 透過陶瓷 frit、微稜鏡層或獨立漫射反射板轉為室內擴散光，對應的冷季能量潛力約為 212.404／424.807／637.211 kWh；暖季皆為 0。

這些數字只能證明「有光能可被接收」，不能證明照度品質。下一階段必須建立：

1. 池畔、救生員、泳道及天花板 sensor grid。
2. 鏡面、玻璃與漫射接收面的 BSDF／可見光透射率與反射率。
3. 無鏡、裸露鏡面、季節遮罩三組年度 Radiance 類比較：lux、時數、均勻度與眩光。
4. 若天花板已足夠明亮，應減少擴散比例，避免把本可進入池面的能量轉走。

DOE指出有效採光必須提供受控制且有品質的照明、真正降低照明需求，並避免過量直射造成眩光與熱增益。[DOE Lighting & Daylighting](https://www.energy.gov/cmei/buildings/zeb-technologies-lighting-daylighting) 鏡面眩光則須按反射照度、面積、距離與觀察點全年評估。[Sandia Solar Glare and Flux Analysis Tools](https://www.sandia.gov/glare-tools/)

## 8. 依基地與建築本體可採用的其他節能策略

### 優先 1：先降低泳池最大熱損失

鏡牆主方案的冷季池面收益約 0.3～0.7 MWh／典型年；室內池真正的大能耗通常來自蒸發、除濕、通風與補熱。應優先納入可收納式池蓋、池水與室溫協調控制、低風速水面氣流、除濕冷凝熱回收及排風熱回收。DOE資料指出商業池蓋可大幅減少蒸發與池水加熱負荷；ASHRAE也要求全年一起評估除濕、池水加熱、風機與熱回收，而非只看太陽得熱。[DOE Pool Efficiency](https://betterbuildingssolutioncenter.energy.gov/sites/default/files/attachments/Pool%20efficiency_.pdf)、[ASHRAE Indoor Swimming Pools](https://handbook.ashrae.org/Handbooks/A19/IP/a19_ch06/a19_ch06_ip.aspx)

### 優先 2：利用 127° 東南向屋面作太陽熱能，而非只依賴鏡面

本基地 TMY 水平面 GHI 約 1,584.747 kWh/m²·yr，負 X 屋面坡向約 127°、屬東南向，具有良好上午太陽資源。4.5° 玻璃屋頂太平且不適合直接承載設備；較好的作法是在結構獨立的 L2 屋面、薄遮簷或服務核心上整合朝南至東南、可維修的太陽熱水／泳池熱能集熱器，並以既有設備作備援。DOE將太陽池水加熱列為可直接抵銷傳統池水加熱能耗的成熟策略。[DOE Solar Pool Heating](https://www.energy.gov/documents/doe-bto-comml-appl-report-full-reportpdf-0)

### 優先 3：把大面積玻璃屋頂分成不同光熱任務

不要讓整片玻璃同時追求最高透光與最低熱增益。建議分成：

- 冷季低角度光的高透射帶。
- 暖季外遮陽／低太陽得熱帶。
- 靠近鏡牆的可控擴散採光帶。
- 可開啟但受濕度控制的高位排熱帶；不能以自然通風取代全年除濕。

外遮陽、方向與玻璃性能必須一起設計；DOE也明確指出建築朝向、玻璃、遮蔭與表面反射共同影響採光和熱取得。[DOE Envelope and Daylighting](https://www.energy.gov/cmei/buildings/zeb-technologies-building-envelope-architectural-considerations)

### 優先 4：讓清水模成為熱穩定層，而不是冷凝面

利用清水模內部熱容量平滑短時太陽得熱，但外側必須有連續保溫、氣密與防潮層，所有內表面溫度需保持在室內露點以上。熱容量不能取代保溫；裸露冷橋與金屬固定件在泳池高濕、氯胺環境中會造成冷凝與腐蝕風險。[ASHRAE Indoor Swimming Pools](https://handbook.ashrae.org/Handbooks/A19/IP/a19_ch06/a19_ch06_ip.aspx)

### 優先 5：延續既有被動水策略

`DEC-036` 的 4.5° 屋頂、被動雨簾、初雨分流、儲水與廁所沖洗已形成水資源回用主線。後續應以台中設計雨量、實際屋頂集水面積與學校沖廁需求定容量，不讓景觀雨簾成為旱天循環耗能設備。

## 9. 建築判斷與未決事項

### 9.1 推薦保留

- L2 水平旋轉 +9.5°。
- 鏡牆向泳池側外傾 +8.5°。
- 鏡面作為冷季太陽導入與清水模立面的精密反差。

### 9.2 分析成立但實體形式尚未核准

- 把鏡面收進季節採光腔，而不是全年裸露。
- 水平百葉 profile cutoff 以 22°為分析起點。
- 東側翼牆／垂直 fins 將有效太陽方位限制於 88°～135°。
- 屋頂靠鏡牆區預留可選擇透射或擴散的材料帶。

這些項目描述的是可達成暖季 0／冷季正收益的工作 solar mask，不代表水平百葉、東側翼牆或季節採光腔已被選為建築方案。

### 9.3 已研究但未選定

- 雙面 360°旋轉葉片：未核准。
- 拉鍊導軌不透光捲屏：有真實產品構件，但未核准。
- 實體摺疊／滑動消光板：有真實產品構件，但未核准。
- 外置百葉／捲門：不作目前方向。
- 使用者已明確表示不滿意上述方案；後續設計重新開放，不得把任一方案當成預設答案。

### 9.4 `OPEN-011` 仍需完成

1. 正式旋轉支點、鏡牆下緣／上緣／牆高與分格。
2. 重新確立季節控制的建築概念；工作 solar mask 仍可作性能比較基準，但不得預設採季節採光腔、旋轉葉片、捲屏或摺板。
3. 鏡面與玻璃的太陽／可見光光學資料、BSDF、污染與老化衰減。
4. 旋轉 L2 與固定玻璃屋頂的完整三維接縫／封閉面，避免把概念縫隙當成光入口。
5. 池面吸收、蒸發、池蓋、除濕熱回收、設備控制與全年熱平衡。
6. 救生員、泳者、樓梯、校園、道路與鄰地觀察點的全年眩光。
7. 專業 Radiance／EnergyPlus 或同等模型，以及材料樣板與等比例節點驗證。

在以上資料完成前，可以說：「PVGIS TMY 工作模型顯示，裸露現行鏡牆會同時增加暖冷季池面能量；加入 22°／88°～135° 建築太陽遮罩後，9 組工作情境暖季增量為零、冷季維持正收益。」不得縮寫成無條件的施工節能保證。
