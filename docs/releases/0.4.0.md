## 已知限制

- 本版是概念設計級 MVP；材質 palette、雨／光效果、鏡牆與 L2 工作高度不代表施工材料或性能，精修由 `TASK-019` 接續。
- Three.js production chunk 約 571 kB（gzip 約 149 kB），Vite 會提出大於 500 kB 的非阻塞警告；後續可依實際載入性能決定是否拆分。
- `OPEN-006`、`OPEN-008`、`OPEN-009`、`OPEN-011`、`OPEN-013`、`OPEN-014` 仍保持未決；Viewer 不代替測量、結構、機電、眩光、熱舒適或施工驗證。
