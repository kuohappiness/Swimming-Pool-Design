- `TASK-016`：把太陽研究的 2F 水平旋轉比較範圍由 −12°～+18° 擴充為 −20°～+40°，維持每格 0.5°。
- 補上 −20°～+40° 的可見刻度與輔助說明；初始值仍由模型的 confirmed `+9.5°` 提供，不把研究滑桿範圍誤述為設計定案。

- 補上水平旋轉及鏡牆外傾的 `aria-valuetext`，讓鍵盤與輔助科技取得和畫面一致的即時角度。
