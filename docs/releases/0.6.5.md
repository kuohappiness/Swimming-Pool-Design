## 修正內容

- L1 Y0 立面改回分段材料：池廳 X0.5～X31 維持安全玻璃，只有服務本體 X31～X39 採自然灰清水模；西端外牆退縮至 X0.5，`EN-01` 同步右移至 X1～X3。
- L1 西端 X0～X0.5 新增傾斜玻璃屋簷與下方雨水回收系統；服務中心後側 X31～X39／Y13.5～Y14.5 補上透明玻璃屋簷。SITE-XY 的 Y0～Y14 格線不變，Y14～Y14.5 明確標為突出屋簷。
- L2 Y0 保持 X29～X41 全寬安全玻璃；Viewer 以淡藍透明雙面材質、反射高光、上下框與豎梃提高從 Y0／Y14 兩側的可辨識度，後方 Y2.5 清水模牆仍是獨立牆面。
- 3F 平面圖將樓板、屋頂、鏡牆與光電排布統一到同一個 +25.5° transform，消除太陽能板的額外旋轉；光電改為淡藍透明。圖集 HTML 直接內嵌 SVG，新增預設勾選的獨立「太陽能板」控制，可完整隱藏／顯示 `RF-PV-RES-01`。
- 縱向剖面補上 L1 X0.5、L2 X29、L2／L3 X41 垂直牆面，並以投影虛線表示右移後的 L1 X1～X3 入口；兩處新增玻璃屋簷與雨水回收關係同步進入圖面。
- 3D Viewer 新增「泳池剖視」：由 Y0 長邊固定觀看、允許稍微向下旋轉，隱藏近側地坪／池壁／立面，顯示透明水體剖面、斜底與 X3 淺端 1.20 m → X28 深端 1.50 m／高差 0.30 m 標示；離開剖視、切換場景或重設時會恢復完整模型與一般相機限制。
- 四張 V065 current SVG／PNG、圖集、3D Viewer、solar-study HTML、canonical model、契約、公開理念、generated data 與版本資料全部同步。
