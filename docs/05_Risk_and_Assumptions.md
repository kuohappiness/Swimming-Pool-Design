# 中師實小游泳池改建｜Risk & Assumption List

## A. Assumptions（目前假設）

| 編號 | 假設 | 說明 |
|---|---|---|
| A001 | 建築外框約 30.0 × 12.5 m | 僅用於 V01／V02 白模型 |
| A002 | 泳池約 20.5 × 7.5 m | 尚未依正式基地尺寸校正 |
| A003 | 二樓樓板高度約 4.2 m | 待空間與結構檢討 |
| A004 | 入口突出量體約 2.2 m | 僅為方案 B 概念尺度 |
| A005 | 優先可配置 3 水道 | 若基地不足則退回至少 2 水道 |
| A006 | 淺端位於入口側、深端位於入口對側 | 須再依現況與教學需求確認 |
| A007 | 景觀走道沿長邊玻璃配置 | 實際寬度與長度待平面設計 |
| A008 | 40 間更衣／淋浴可集中於入口側二樓 | 待實際排入驗證 |
| A009 | 玻璃屋頂可作為主要採光來源 | 尚未做熱環境、遮陽與冷凝分析 |

## R. Risks（風險）

| 編號 | 風險 | 影響 | 對策 |
|---|---|---|---|
| R001 | 基地尺寸不準 | 可能導致整體配置重做 | 優先取得正式丈量或 CAD |
| R002 | 40 間更衣／淋浴空間不足 | 可能壓縮走道、廁所或泳池 | 先建立最小合理單元並試排 |
| R003 | 屋頂 10° 導致低側淨高不足 | 影響泳池空間、玻璃與結構 | 依最新紅線重新計算高低點 |
| R004 | 大面積玻璃屋頂造成過熱 | 影響舒適度與能耗 | 後續評估遮陽、半透明比例及排熱 |
| R005 | 室內泳池高濕造成冷凝 | 影響玻璃、鋼構與維護 | 機電除濕與結露分析需專業介入 |
| R006 | 全玻璃長邊造成眩光或曝曬 | 影響教學與使用 | 後續評估遮陽、Low-E 玻璃與方向 |
| R007 | 靠窗樓梯與玻璃立面衝突 | 可能影響逃生、結構與維護 | 平面深化時同步檢討 |
| R008 | 單一主樓梯可能不符逃生需求 | 可能需要第二逃生動線 | 由建築師依實際法規檢討 |
| R009 | 泳池面積最大化壓縮池岸安全空間 | 影響教學、安全與維護 | 設定池岸最小寬度後再最大化泳池 |
| R010 | 女廁 5 間與其他空間衝突 | 可能需減量 | 完成一樓配置後檢核 |
| R011 | DXF 與 HTML 不同步 | 造成設計判讀錯誤 | 使用共用參數與版本檢查表 |
| R012 | HTML 在手機附件預覽無法執行 | 使用者無法操作 3D | 透過 GitHub Pages 網址開啟 |
| R013 | 概念模型被誤用為施工資料 | 產生設計與責任風險 | 所有文件標示「暫估／概念設計」 |
| R014 | 法規、結構、消防、無障礙未檢討 | 方案可能無法落地 | 概念成熟後交由合格專業人員深化 |
