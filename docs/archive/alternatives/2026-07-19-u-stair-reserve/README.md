# 備用方案：入口側折返樓梯與未來景觀走道

- 記錄日期：2026-07-19
- 狀態：備用方案／未納入現行模型

## 1. 設計前提

1. 一樓真正入口位於泳池大廳與戶外前場交界的短邊側。
2. 入口右側紅框範圍是完全戶外區，不配置入口室、樓梯或其他室內空間。
3. 樓梯一樓起跑處直接位於泳池大廳內，不另設樓梯前室。
4. 藍色區域定義為純景觀走道，不是主要動線；現階段只保留未來增設條件。

## 2. 方案內容

本方案採入口側 U 形折返樓梯：

- 第一段樓梯由一樓入口附近向泳池方向上升。
- 於约 `+2.25 m` 到達中繼平台後轉向 180°。
- 第二段樓梯折返回入口側，於 `+4.50 m` 到達二樓。
- 二樓出口再以約 3 m 的短走道銜接既有男女分流區。
- 中繼平台向泳池方向只保留未來景觀走道的結構接口。

## 3. 尺寸基礎

| 項目 | 尺寸／結果 |

| 樓層高度 | 4.50 m |
| 總階數 | 30 階 |

| 每梯段 | 15 階、14 個踏面 |
| 單梯段水平長度 | 4.20 m |
| 樓梯淨寬 | 1.80 m／梯段 |
| 中繼平台標高 | +2.25 m |
| 樓梯坡度 | 約 26.6° |
| 折返梯淨占用 | 約 6.00 × 3.60 m |
| 加計梯梁、扶手後 | 約 6.20 × 4.00–4.20 m |

上述樓梯比例在概念階段具合理性，但仍須由建築師與結構、消防專業依實際用途及當地法規完成正式檢核。

## 5. 真實執行條件

目前泳池兩側長邊池畔各約 3.0 m，而本方案完成寬度約 4.0–4.2 m，因此不能直接放入既有斷面。

可行條件依優先順序如下：

1. 在約 `x=18–24` 的入口側長邊設置局部室內樓梯灣，外牆至少向外增加約 1.0–1.2 m。此尺寸只足以容納樓梯本體。
2. 若要求樓梯旁另保留約 1.2 m 的獨立池畔通道，總需求寬度將達約 5.2–5.4 m；需擴大更多，或在詳細設計中驗證能否安全通過較高的回程梯段下方。
3. 不建議為了塞入現有 3.0 m 池畔帶而將每段樓梯由 1.80 m 縮至約 1.35–1.40 m。
4. 不占用完全戶外前場，也不以移動入口作為本方案成立條件。

## 6. 中繼平台與斜屋頂

中繼平台必須靠近斜屋頂高點：

- `x=18` 附近屋頂標高約 `+4.421 m`，平台上方理論淨高約 2.171 m。
- `x=17` 附近屋頂標高約 `+4.343 m`，平台上方理論淨高約 2.093 m。
- 上述數值尚未扣除玻璃屋頂梁、接頭及裝修厚度。

因此平台宜控制在 `x≈19` 附近，並於後續設計確認局部挑高、屋頂開口或梁位調整，不能再明顯向泳池低屋頂方向移動。

## 7. 未來景觀走道

藍色景觀走道不列入本階段必要功能，只保留中繼平台端的結構與欄杆銜接條件。

- 以目前 4.5° 斜屋頂計算，若維持 2.10 m 理論淨高，自 `x=19` 向泳池方向最多只能延伸約 1.91 m。
- 扣除屋頂構造後，概念上較安全的短走道長度約 1.2–1.5 m。
- 若未來希望設置圖示中的長距離景觀走道，必須同步抬高或重做玻璃斜屋頂，並重新審查結構、欄杆、維護與避難條件。

## 8. 啟用本備用方案前的決策

本方案只有在確認下列條件後才可轉為現行方案：

1. 入口側長邊是否允許局部向外增加至少 1.0–1.2 m。
2. 一樓池畔通路是否允許於樓梯區局部中斷，或可在回程梯段下方維持合格淨高與通行寬度。
3. `x≈19` 的屋頂與二樓樓板是否允許為樓梯平台及回程梯段調整開口和梁位。
4. 二樓是否可保留約 3 m 的銜接走道通往既有男女分流區。

在上述條件未確認前，本資料夾只作為設計比較與未來重啟依據，不應直接驅動現行模型。
