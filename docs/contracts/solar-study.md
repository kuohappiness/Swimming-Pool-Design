# 太陽研究契約

- 類型：output-contract
- 狀態：active
- Owner：[05｜模型契約](../05_MODEL_CONTRACT.md)

## 必要行為

`reference/solar-study/index.html` 是概念教育展示，必須：

- 從 `model/project-model.json` 讀取校址座標、時區、建築長軸、模型版本與專案名稱。
- 以同一太陽位置函式依日期、時間、經緯度與 UTC offset 計算高度角及方位角。
- 左側平面固定 1F，只旋轉 2F 更衣量體；右側剖面只調整面池端牆外傾。
- 讓季節、時間、2F 水平旋轉及牆面外傾共同更新反射結果。
- 清楚區分操作預設與正式設計值，並將未定性能連結 `OPEN-011`。
- 支援 320 px 以上手機、鍵盤操作、可讀 SVG title／description，且無外部執行依賴。
- 明示結果是幾何概念判讀，不是眩光、材料、遮蔭、熱負荷或水溫模擬。

## 方位不變條件

依 `DEC-029`，固定建築與 2F 的基準 transform 必須使用模型本地 +X 方位 307°。面池反方向 127° 只可用於鏡牆法線、目標方向或反射判讀，不得作為整棟平面基準旋轉。

目前平面仍有 180° 首尾顛倒，由 [TASK-001](../07_ACTIVE_WORK.md) 管理。

## 驗收

- `REF-001` 與太陽研究的核心端、泳池遠端、真北及 2F 所在端一致。
- 冬至 09:00 操作預設能導向泳池；夏至 09:00 同一預設不命中泳池。
- 反射函式涵蓋方位環繞、背光、平面偏差與向下角度測試。
- build 產生 `dist/reference/solar-study/index.html`，桌面與手機 smoke 通過。
