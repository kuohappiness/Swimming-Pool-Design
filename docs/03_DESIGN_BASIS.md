## 3. 空間與建築意圖

- 泳池大廳為主要挑高空間；玻璃屋頂只覆蓋未被 L2 擴建量體遮蔽的區域。
- 玻璃屋頂由泳池遠端低側上升至 L2 擴建量體邊緣高側，平面投影不進入擴建量體下方。
- `REF-401` 的玻璃屋頂以高端靠近 L2 樓板的概念關係顯示；10° 維持 confirmed，正式高低標高仍為 deferred。
- `EXT-L2-01` 低 X 面池端牆是鏡面反射牆，牆頂向泳池側外傾；正式角度、牆高與性能仍未定。
- L1 服務核心維持原廁所基地 11.0 × 13.5 m；L2 另向泳池大廳上方伸出 5.0 m，形成 16.0 × 13.5 m 工作外框。
- L2 擴建量體只存在於二樓；其下方不新增 L1 房間，保持泳池大廳／池畔開放空間。
- 長向外牆以玻璃採光及通透視線為重要設計語彙。
- 黃色泳池基地與綠色既有廁所基地合併規劃；原廁所拆除重建。
- `EN-01` 位於兩基地交界的校園／操場到達側，是進入 L1 戶外前場的日常到達入口；它不是室內共用前室的入口。
- 從 L1 戶外前場分別以三個彼此獨立的開口進入泳池大廳、男廁與女廁；除已確認的廁所前後門外，其他門若出現，只能標示為逃生、設備、維修或後勤用途。
- 建築與基地按真北放置，不以畫面上方取代方位資料。

`ST-01` 是 A2 兩段同向懸空式樓梯：共用上樓，L2 才分流進入男女區。樓梯設於長向玻璃牆的乾式透明樓梯廊，採雙側厚鋼梯梁與透明欄杆；中間平台懸浮，梯下完全開放且不封成水泥量體。樓梯結構不得懸掛於玻璃屋頂。

- L1 男女廁採本地座標男左女右：男廁位於較低 X 側，女廁位於較高 X 側。
- 操場側綠框是戶外空間，不是共用前室；泳池大廳、男廁與女廁各自以獨立開口直接面向這個戶外空間。
- 男女廁各有一樘朝向操場側戶外空間的前門，以及一樘連接泳池後側乾式通道的後門；後門各自管制，非泳池使用時段關閉。
- 泳池使用者可由泳池大廳經最上方的連續乾式通道，再由各自後門進入男廁或女廁；`REF-101` 必須完整畫出泳池大廳與這條通道的連接，不能只留下文字標籤。
- 同一廁所的前、後門不得在同一直視軸上，入口以錯位及短遮擋牆維持隱私。
- L1 廁所使用者不經 `ST-01`；`EN-01` 門檻仍與 `O-SITE-01` 共點，兩者共用的本地原點必須精確維持 `[27, 0, 0]`。到達路徑 `RTE-L1-ARRIVAL-01` 必須由該門檻出發、完整留在戶外前場 bounds 內，並在樓梯開始前轉至其外側，以正淨空通往戶外前場。樓梯不得與這條路徑相切或重疊，也不得阻斷三個獨立開口、乾式通道或廁所前後門動線。

## 6. 太陽反射研究基準

- 1F 地基、泳池本體與長邊玻璃牆是固定基準，不納入旋轉變因。
- 2F 更衣量體可在平面水平旋轉；面向泳池的端牆可另在剖面外傾。兩個角度分別控制反射方位與向下角度。
- 設計目標是冬季 08:00～10:00 增加朝泳池方向的反射，夏季上午不增加、最好偏離泳池的反射。
- 優先以建築本體量體與牆面完成季節選擇性，不新增額外反射層板。
- 固定建築與 2F 量體的基準 transform 使用本地 +X 307°；面池反方向 127° 只作鏡牆法線、反射目標或判讀，不得再旋轉整棟平面。
- 互動展示的 `+4.5°` 水平旋轉與 `9.5°` 外傾只是操作預設，用來示範冬季命中、夏季避開；不是已確認的建築尺寸，也不得寫入 L2 正式幾何。
- 最終旋轉支點、精確角度、牆高、材質反射率、眩光、遮蔭與熱效益由 `OPEN-011` 管理。
