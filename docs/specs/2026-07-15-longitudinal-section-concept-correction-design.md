## 1. 目的

修正 `REF-401` A–A 縱剖面目前對屋頂高度、L1 入口戶外區及 L2 面池端鏡牆的錯誤表達。修正版必須讓三者的空間關係一眼可辨，同時避免把使用者的示意紅線及「一些傾斜」轉寫成不存在的精確標高或角度。

本輪只建立可信的概念關係。屋頂交界施工細節、入口戶外區精確尺寸、鏡牆角度與性能繼續由既有 OPEN 管理。

## 2. 新來源證據

使用者提供的標註圖在實作時以原始位元組登錄為：

- ID：`SRC-CONCEPT-009`
- 路徑：`source-materials/concepts/SRC-CONCEPT-009_longitudinal-section-correction-annotated.png`
- 收件日期：2026-07-15
- 像素：2216 × 1130
- 檔案大小：312250 bytes
- SHA-256：`3CD710CEC62E32F2209EFA731FCF0EBFDA38A978BD0925A504481EE563175034`

可直接觀察到的標註如下：

1. 紅線位於現有玻璃屋頂下方，高端接近 L2 樓板位置。
2. 綠框圈出 L1 服務核心右端目前以灰色實體表示的區域。
3. 使用者文字指出 `EXT-L2-01` 的外牆是先前討論的鏡面反射牆，並應帶有傾斜。

設計解讀如下：

1. 紅線確認屋頂應整體下移的概念關係，不確認任何精確標高。
2. 綠框區應表達為入口戶外區，不是室內服務核心量體；精確寬度仍未確認。
3. 鏡牆是 `EXT-L2-01` 面向泳池的低 X 端牆；牆頂向泳池側外傾。外傾程度、牆高與反射性能仍未確認。

## 3. 核准的空間關係

### 3.1 玻璃屋頂

- `RF-GL-01` 維持已確認的 10° 單坡，從泳池遠端低側上升至 L2 擴建量體邊緣高側。
- `REF-401` 的概念高端應靠近 L2 樓板，不再靠近 L2 量體頂部。
- 低端依同一 10° 坡度同步下移，不另外硬編碼第二個高度。
- 圖面必須標示「位置示意；標高／交界待 `OPEN-010`」，不得顯示假精確標高。
- `J-RF-L2-01` 仍是 deferred 交界；玻璃屋頂不承載 `EXT-L2-01`。

### 3.2 L1 入口戶外區

- 綠框位置沿用 `DEC-028`，表達為 `Z-L1-ENTRY-01` 操場側入口戶外區。
- 剖面中移除該區的灰色室內量體填色，以開放空間語彙、綠色虛線範圍及文字標籤表達。
- L2 量體可位於入口戶外區上方；圖面不得因有上方樓板而把 L1 區域畫成封閉房間。
- `Z-L1-ENTRY-01` 的名稱應從「共用前室」修正為「操場側入口戶外區」。其戶外語意是 confirmed，精確幾何仍由 `OPEN-008` 管理。

### 3.3 L2 面池端鏡牆

- 新增穩定立面 ID `F-MIR-01`，表示 `EXT-L2-01` 面向泳池的低 X 端鏡面反射牆。
- `REF-401` 將 `EXT-L2-01` 由矩形改為帶外傾端面的概念多邊形；牆頂向泳池側偏移。
- 鏡牆使用與一般紅色量體不同的藍灰反射材質語彙，標註「鏡面反射牆／外傾示意」。
- 圖面不標外傾角，不使用太陽研究頁的 9.5° 操作預設作為正式建築幾何。
- 正式外傾角、水平旋轉、牆高、材料反射率、眩光、遮蔭、結構及熱效益繼續由 `OPEN-011` 管理。

實作依 repository owner 規則按下列順序同步：

3. `docs/03_DESIGN_BASIS.md`：記錄屋頂高端接近 L2 樓板、入口戶外區及面池端鏡牆的當前空間意圖，不新增精確數值。
4. `docs/05_MODEL_CONTRACT.md`：加入 `F-MIR-01` 與 confirmed 語意／deferred 幾何的狀態規則。

6. `docs/07_ACTIVE_WORK.md`：維護 `TASK-006` 狀態，並讓 0.2.0 發布工作等待本任務完成。
7. `model/project-model.json`：登錄來源與 `F-MIR-01`，修正 `Z-L1-ENTRY-01` 名稱及 sheet references；屋頂標高仍為 null/deferred，不新增鏡牆角度值。

模型只保存已知語意、狀態、來源及穩定 ID。`REF-401` 可使用明確標為 conceptual/deferred 的顯示幾何，但不得讓 consumer 把它讀成正式尺寸。

## 5. `REF-401` 圖面設計

由左至右的閱讀順序保持泳池遠端、玻璃屋頂、樓梯、L2 擴建及入口端：

1. 將 `RF-GL-01` 整體下移，高端顯示在 L2 樓板附近；10° 標註沿屋頂保留。
2. 在屋頂高端與 L2 端牆附近保留 `J-RF-L2-01` deferred 標示，但避免文字壓住鏡牆。
3. 將 `EXT-L2-01` 面池端改成向左外傾的鏡面邊界，量體本身仍使用紅色填色。
4. 將右下入口戶外區從灰色 `CORE-01` 填色中視覺挖空，改用綠色虛線框與 `Z-L1-ENTRY-01 入口戶外區` 標籤。
5. 保留 `ST-01`、L2 +3.600、屋頂平面跨度 19.0 m 及 L2 16.0 m 等既有有效資訊。
6. 不在此圖加入太陽光線、反射路徑、9.5° 或其他研究頁控制值。

## 6. 實作邊界

本任務包含 owner 文件、來源登錄、單一模型、`REF-401` renderer、樣式、sheet references、測試與視覺驗證。

本任務不包含：

- 關閉 `OPEN-008`、`OPEN-010` 或 `OPEN-011`。
- 決定入口戶外區寬度、屋頂高低標高或鏡牆正式外傾角。
- 修改太陽研究的操作預設或反射演算法。
- 設計鏡牆構造、材料規格、眩光控制或屋頂交界施工細節。
- 因本張剖面而重畫其他參照圖；其他圖若出現契約矛盾，只記錄為缺陷或後續工作。
