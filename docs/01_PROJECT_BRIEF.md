# 01｜專案簡報

## 1. 專案目的

本專案為國立臺中教育大學附設實驗國民小學既有游泳池與相鄰廁所基地的翻修概念設計。目標不是直接產出施工圖，而是建立一套可理解、可追溯、可即時檢視且能隨新資料修正的數位規劃模型。

第一個正式成果是「空間參照圖集」：先用穩定 ID、樓層、格網與共同尺寸讓討論指向同一位置；後續可信即時 3D Viewer 再直接沿用相同模型資料。

## 2. 規劃範圍

- 黃色標示的既有泳池基地。
- 綠色標示的既有廁所基地；現況設施拆除重建並納入泳池整體計畫。
- 泳池大廳、池體、池畔、服務核心、廁所、更衣淋浴、入口與主要樓梯。
- 建築外殼、長邊玻璃牆、10° 單坡玻璃屋頂、主要動線、量體及基地方位。
- 由單一模型產生的參照圖集、即時 3D Viewer 與未來 DXF 概念輸出。

## 3. 核心需求

- 泳池配置三水道，水深由淺端 1.2 m 連續變化至深端 1.5 m。
- 玻璃屋頂只覆蓋未被二樓擴建量體遮蔽的泳池大廳；二樓擴建端為高側，泳池遠端為低側。
- `EN-01` 是唯一日常人員入口。
- `ST-01` 沿長邊玻璃牆設於乾式透明樓梯廊，採 A2 兩段同向、懸浮中間平台、雙側厚鋼梯梁、透明欄杆及完全開放梯下。
- L1 重建男女廁所並採男左女右；操場側以有遮蔽共用前室分流，泳池側後門只在泳池使用時段開啟。
- L2 由原服務核心加上向泳池大廳上方伸出的二樓專用擴建量體組成；擴建量體下方的 L1 保持開放，男、女更衣淋浴區在中央樓梯落點後向左、右嚴格分流。
- 男女各 15 間更衣＋淋浴整合單元，空間允許時各擴充至 20 間。
- 每間單元內設壁掛置物櫃，不設集中式置物櫃區。
- 方位是模型的一級資料，所有圖面與 Viewer 都能辨識真北。

## 4. 工作前提

- 本案不等待正式丈量或 CAD 才開始。
- 現有衛星比例尺、影像、手繪圖與使用者決策形成明確規劃值。
- 畫面直接顯示數值，不在每個尺寸旁附加「約」或「暫估」。
- 每個數值在資料層保留來源、狀態與修訂，使未來可替換而不必重做成果。

## 5. 非本階段範圍

- 施工、結構、機電、消防、無障礙或法規簽證。
- 工程數量、預算、招標及工期承諾。
- 以概念模型宣稱現況已經正式測量。
- 將歷史 V02／V03 成果直接視為當前模型真相。

## 6. 成功條件

1. 一份機器可讀的單一模型驅動六張參照圖，之後可延伸至 3D Viewer 與 DXF。
2. 手機及桌面能切換圖面、縮放並點選 ID 查看資料。
3. 北向、比例、模型版本、狀態及主要尺寸清楚可見。
4. 男女各 15＋5 單元、入口、樓梯與屋頂規則都有自動驗證。
5. 修改參數後，建置程序能證明所有成果仍使用同一份資料。
