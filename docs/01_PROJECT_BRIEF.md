# 01｜專案簡報

## 1. 專案目的

本專案為國立臺中教育大學附設實驗國民小學既有游泳池與相鄰廁所基地的翻修概念設計。目標不是直接產出施工圖，而是建立一套可理解、可追溯、可即時檢視且能隨新資料修正的數位規劃模型。

第一個軟體成果是「可信的即時 3D Viewer」：每次修改設計參數後，都能立即在瀏覽器看到結果，並能辨識目前版本、方位、尺寸來源與尚未決定的部分。

## 2. 規劃範圍

本案納入：

- 黃色標示的既有泳池基地。
- 綠色標示的既有廁所基地；現有設施視為拆除重建，新的空間可整合進泳池計畫。
- 泳池大廳、池體、池畔、服務空間、廁所、更衣與淋浴機能。
- 建築外殼、採光、屋頂、主要動線、量體及基地方位。
- 可由單一參數模型產生的 3D Viewer 與 DXF 概念輸出。

## 3. 核心需求

- 泳池以三水道為優先，基地限制下至少保留二水道。
- 池水深度由淺端 1.2 m 連續變化至深端 1.5 m。
- 泳池大廳為主要挑高空間。
- 玻璃屋頂只覆蓋泳池大廳，不覆蓋更衣、廁所或其他服務空間。
- 長向立面以玻璃與視線穿透為重要設計語彙。
- 男女使用容量各 20 人；確切更衣、淋浴配置在後續空間決策中完成。
- 廁所工作需求為男廁 3 座坐式馬桶、3 座小便斗及洗手台；女廁 5 座坐式馬桶及洗手台。
- 方位是模型的一級資料，所有圖面與 Viewer 必須可辨識北向。

## 4. 工作前提

- 本案不等待正式丈量或 CAD 才開始。
- 現有衛星比例尺、影像、手繪圖與使用者決策可形成明確的規劃數值。
- 畫面與輸出直接顯示數值，不在每個尺寸旁附加「約」或「暫估」。
- 每個數值仍須在資料層保留來源、依據、狀態與修訂紀錄，讓未來可替換而不必重做整套成果。

## 5. 非本階段範圍

- 施工、結構、機電、消防、無障礙或法規簽證。
- 工程數量、預算、招標及工期承諾。
- 以推測補完尚未決定的入口、樓梯或更衣室配置。
- 將歷史 V02／V03 成果直接視為當前模型真相。

## 6. 成功條件

專案在下列條件同時成立時，才可稱為第一階段成功：

1. 一份機器可讀的單一模型資料可驅動 Viewer 與 DXF。
2. Viewer 可立即載入、旋轉、縮放、切換圖層並回到標準視角。
3. 北向、比例、模型版本及主要尺寸清楚可見。
4. 尚未決定的空間不被偽裝成已確認設計。
5. 修改參數後，可用自動檢查證明畫面與輸出仍使用同一份資料。
